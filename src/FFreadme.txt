# FreeFEM++ - Installation Guide for Windows

## Introduction

Welcome to the installation guide for FreeFEM++ on Windows using the precompiled package. This README file provides detailed instructions on how to download, install, and configure FreeFEM++ on your Windows system.

## Prerequisites

Before getting started, make sure you have the following prerequisites on your system:

- Windows operating system (Windows 7 64 bit or later)
- Internet connection (for downloading the package)

## Installation

1. **Download the precompiled package:**
   Visit the official FreeFEM++ website at [https://doc.freefem.org](https://doc.freefem.org/introduction/download.html) and download the latest version named [64-bit Windows].

2. **Extract the package:**
  Double-click on the downloaded .exe file.

3. **Configure the environment:**
   Follow the normal Windows installation procedure, accepting the default values if you do not want to give the programme a specific destination folder, BUT MAKE SURE YOU HAVE CHECKED THE CHECK BOX that reads [Add application directory to your system path (if missing you can have trouble with on-thr-fly graphics)].
When finished, click on [install] and wait for the installation to complete.  

4. **Verify the installation:**
   Open a new Command Prompt window and execute the command:
   FreeFem++
   If the programme is correctly installed, a short summary with the programme version and help will be displayed.

## Running FreeFem++ scripts

   Open the folder where the FreeFem++ script you want to run is located, which you remember has the extension .edp [e.g. VIA-american_r2.edp] and double-click on it.
